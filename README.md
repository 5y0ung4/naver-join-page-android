# naver-join-page-android

네이버 구버전 모바일 UI를 레퍼런스로 한 **Android(Kotlin) 클론 학습 프로젝트**입니다.

### purpose
- XML 기반 Android UI 구성 연습
- Retrofit + Coroutine을 활용한 서버 통신
- 요청/응답 데이터 모델링(Data Class)
- Activity 중심 구조에서 ViewModel 분리(MVVM 기초)
